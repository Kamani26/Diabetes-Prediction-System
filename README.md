# 🏥 Diabetes Prediction System

A Machine Learning web app that predicts whether a patient 
is diabetic or not based on medical parameters.

## 🎯 Accuracy
- Model: Random Forest Classifier
- Accuracy: 88.31%

## 🛠️ Technologies Used
- Python
- Scikit-learn
- Pandas
- Streamlit

## 📊 Dataset
- Pima Indians Diabetes Dataset (Kaggle)
- 768 patients, 9 features

## 🚀 How To Run
streamlit run app.py
